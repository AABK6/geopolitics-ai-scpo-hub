# PRISM slide assets

Downloaded for Module 03 archival use.

- `prism-documentcloud-guardian.pdf`: DocumentCloud copy embedded by The Guardian, 11 pages.
  Source: https://www.documentcloud.org/documents/813847-prism/
- `documentcloud-pages/`: PNG renders of the DocumentCloud PDF, 1584 x 1224.
- `PRISM_Powerpoint_Slides_re_Data_Acquisition_ACLU.pdf`: ACLU-hosted PRISM PowerPoint slide PDF, 4 pages.
  Source: https://www.aclu.org/documents/prism-powerpoint-slides-re-data-acquisition-2
- `aclu-pages/`: PNG renders of the ACLU PDF.

Use the DocumentCloud pages for the fuller Guardian set, and the ACLU pages when the shorter FOIA-style set is preferable.
