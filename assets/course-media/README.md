# Course media assets

This directory holds candidate media for the primer module pages. It is organized by module so assets can be inserted later without changing the course text.

## Structure

- `module-01/clinton-sais-2000/`: archival Clinton video for Module 01.
- `module-02/great-firewall/`: final Great Firewall teaching illustration for Module 02.
- `module-03/prism/`: PRISM PDFs and rendered slide PNGs for Module 03.
- `module-07/ideological-frames/`: final Imagen illustration for the ideological frames module.
- `module-08/governance-axes/`: five-axis governance teaching diagram for Module 08.
- `_drafts/illustrations/`: superseded or alternate illustration attempts kept for reference.

Prefer the module folders for production page inserts. Use `_drafts` only if you want to compare or recover an earlier version.
